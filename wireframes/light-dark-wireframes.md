# Wireframes / UI Spec

## Splash/Auth
- Light: white background, light-purple logo and CTA.
- Dark: charcoal background, purple accent logo and fields.

## Tab 1 Device Control
- Collapsible Application Control card with search + bulk controls.
- Restriction toggles card with confirmation sheet for destructive actions.
- Password management card for change/enforcement mode.

## Tab 2 Firewall
- Rules list with priority chips.
- Add-rule sheet (domain wildcard + app target).
- Real-time blocked attempts feed.

## Tab 3 Settings
- Theme mode segmented control.
- Password policy and reset controls.
- VPN policy import/export and retention controls.
- About/legal/support section.
